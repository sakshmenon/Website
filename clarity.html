    Clarity is an advanced object detection system designed to assist visually impaired users in their daily activities, particularly in grocery stores and restaurants. The project aims to keep about a 90% success rate in object identification and ensure compliance with WCAG (Web Content Accessibility Guidelines) for accessibility. This involves using state-of-the-art machine learning algorithms and incorporating user feedback to continually improve the system's accuracy and usability. Clarity aims to mitigate challenges faced by visually impaired individuals by leveraging cutting-edge machine learning and AI technologies to create an intuitive and user-friendly object detection system. This system will empower visually impaired users to interact with their environment more effectively and make informed decisions independently. Our project will create a user-friendly platform to assist them by identifying objects and reading text aloud in real-time. This tool aims to enhance spatial awareness and independence through audible feedback.
