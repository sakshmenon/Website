    • Covid 19 Detection

    Image Classificaiton Appliction

    Software Used

    Python - TensorFlow
    Jupyter Notebook

    Tools

    Numpy
    OpenCV
    Git

    The purpose of this project is to demonstrate the use of tensor flow based CNN layers and ResNET models in performing image classification between 3 different classes of lung X-ray images. These classes are as follows: Normal Lungs, Covid 19 Lungs, and Viral Pneumonic Lungs. To achieve success in this classification, X-ray images are combined with their provided masks to tune out irrelevant information. The preprocessed images are them used to train the model after which predictions are made.
